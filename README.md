# Formative 3 — Probability, Bayesian Inference, and Gradient Descent

## Structure
- `part1_em/` — EM algorithm (heights mixture model)
- `part2_bayes/` — Bayesian sentiment analysis (IMDb)
- `part3_manual_gd/` — Manual gradient descent calculations
- `part4_gd_code/` — Gradient descent implementation in code

## Setup
1. Clone the repo
2. Download datasets (see below) and place in `data/`
3. `pip install -r requirements.txt`

## Dataset links
- Heights: https://www.kaggle.com/datasets/jacopoferretti/parents-heights-vs-children-heights-galton-data
- IMDb reviews: https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews
  → download `IMDB Dataset.csv`, save as `data/imdb_reviews.csv`
